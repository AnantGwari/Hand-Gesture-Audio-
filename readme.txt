Project - Hand gesture audio controller
Author - Anant Gwari (https://github.com/AnantGwari)
Language - Python 3.9
Video Demonstration-->
https://youtu.be/L0RzH93SMBs

The following is a python script utilizing the opencv, mediapipe and pycaw module. The program reads a video input from the system's default videocam and uses the mediapipe module to detect hands and recognize the 21 importants hand points. Then it calculates the distance between the index finger and thumb and changes the master volume of the device in accordance to the distance.
